# 淫纹

:::tip插件信息
- 插件位置：[plugins/YinWen.py](https://github.com/phantom-sea-limited/Bot/blob/main/plugins/YinWen.py) [plugins/__face.py](https://github.com/phantom-sea-limited/Bot/blob/main/plugins/__face.py)
- 插件作者：[Rcrwrate](https://github.com/Rcrwrate)
- 插件类别：整活
- 频率限制：<span id="green">NO</span>
- 可扩展性：<span id="green">YES</span>
- 更新状态：<span id="green">已完工</span>


- 触发方式：
  - 群聊消息
- 输出类型：文字
:::

### 功能说明

任何一条群聊消息都有概率触发，随机刻印淫纹（夹带私货警告）或随机返回一个颜文字（X岛匿名版上扒下来的）

淫纹刻印最多一次/天/人，颜文字触发不设限

### 扩展

你可以手动添加更多的随机事件，直接在函数末尾直接续写即可
